1. angular.json
Contains Angular project configuration such as build, serve, assets, styles, and scripts.

2. tsconfig.json
Main TypeScript configuration file used by the Angular project.

3. tsconfig.app.json
TypeScript configuration specifically for the application source code.

4. package.json
Stores project information, dependencies, scripts, and package versions.

5. src/main.ts
Entry point of the Angular application. It bootstraps the root application.

6. src/app/app.config.ts
Configures the standalone Angular application, including providers and routing.

7. src/app/app.ts
Root standalone component of the application.

8. src/index.html
Main HTML page loaded by the browser where the Angular app is rendered.
