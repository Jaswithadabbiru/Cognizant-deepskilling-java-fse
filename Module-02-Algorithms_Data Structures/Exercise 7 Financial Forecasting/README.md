\# Exercise 7 - Financial Forecasting



\## Objective



Predict the future value of an investment using recursion.



\---



\## Formula



Future Value = Present Value × (1 + Growth Rate)^Years



\---



\## Concepts Used



\- Recursion

\- Base Case

\- Recursive Call



\---



\## Time Complexity



O(n)



The recursive method makes one recursive call for each year.



\---



\## Space Complexity



O(n)



The recursive call stack stores one function call per year.



\---



\## Optimization



Instead of recursion, an iterative loop or the mathematical formula can be used to avoid recursive function calls and reduce stack usage.



\---



\## Sample Output



Present Value : ₹10000



Growth Rate : 10%



Years : 5



Future Value : ₹16105.10
