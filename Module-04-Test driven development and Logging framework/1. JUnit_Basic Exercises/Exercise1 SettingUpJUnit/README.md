# Exercise 1 - Setting Up JUnit

## Objective
Set up JUnit in a Maven project and execute a basic unit test.

## Technologies Used
- Java
- Maven
- JUnit 4.13.2
- IntelliJ IDEA

## Project Structure

src/

│   └── Calculator.java

└── CalculatorTest.java

## Files

### Calculator.java
Contains a simple calculator class with an add() method.

### CalculatorTest.java
Contains a JUnit test case to verify the add() method.

## Result
The JUnit test executed successfully and all test cases passed.
